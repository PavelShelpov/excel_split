Задача: разработать программу для разделения файла Excel на разные файлы исходя из категории.
Дано: Excel файл формата .xlsx, .xlsm
	1. На каждом листе программа должна найти начало таблицы с данными и определить наименования колонок.
		Это не всегда первая строка. Первые строки могут быть пустыми или содержать служебную информацию.
		Может быть не определена как форматированная таблица, поэтому искать нужно эвристически.
		Таблица должна быть найдена полностью, с 1 ячейки заголовков до последней ячейки с данными, учитывая, что несколько строк данных могут быть пропущены (пробелы в таблице).
		Координаты начала таблиц на каждом из листов необходимо запомнить для того, чтобы не искать ее вновь в результирующем файле.
	2. Определить наименования колонок, которые встречаются на всех листах (пересечение).
		Чтобы была возможность в последующем отфильтровать каждый из листов по одной и той же категории без конфликтов.
	3. Предложить пользователю выбор колонок (по категориям из которых будет выполняться разделение).
		Если выбрано несколько колонок, то деление будет производится по комбинации категорий.
		Необходимо проверить, что колонка(-и) действительно содержит(-ат) категории (до 50 в сумме), иначе выдать предупреждение.
	4. Предложить пользователю выбрать место для сохранения результирующих файлов.
	5. Дать пользователю информацию о том, сколько файлов будет создано в итоге (уникальных категорий или комбинаций категорий).
	6. Создать полную копию исходного файла.
		Важно! Необходимо именно создать копию, и в последующих шагах работать именно с открытой копией. Перенос форматов в чистый файл недопустим, т.к. это существенно усложняет процесс.
	7. Назвать файл: первые буквы каждого слова названия категории n выбранного столбца 1 + название категории m столбца 2 + ... + название оригинального файла.
		Например: столбец 1 (Город) - категория 1 (Москва), столбец 2 (Улица) - категория 1 (Михаила Ковалева), исходный файл (Продажи_2025-13-09) => итоговый файл (М_Михаила Ковалева_Продажи_2025-13-09.xlsx)
	8. Отфильтровать каждую идентичную таблицу на каждом из листов по полученной из столбца категории (комбинации категорий).
		Важно! Фильтруется только найденная таблица, остальные данные на листе (включая форматы) остаются должны оставаться нетронутыми.
		Если на каком то из листов нет данных с текущим фильтром, то лист удалить.
	9. Сохранить итоговый файл в выбранную пользователем папку.
	10. Повторить начиная с п.5 для каждой следующей категории (комбинации категорий).
	11. Выдать пользователю сообщение об успешном завершении операции или об ошибке.
